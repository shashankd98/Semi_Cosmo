These files contain the data for generating figure 9 of Dattathri & Sharma (2021). There are 7 .dat files, one for each time in the figure. Each file contains a 2D array containing 7 rows:
Row 1: r/R200c
Row 2: density (mp/cm^3)
Row 3: pressure (dyne)
Row 4: t_cool/t_ff
Row 5: velocity (km/s)
Row 6: temperature (T)
Row 7: Mdot (Msun/yr)
